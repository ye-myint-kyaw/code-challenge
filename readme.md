# 99Tech Code Challenge

This repository contains my solutions for the 99Tech take-home code challenge.

## Problems

### Problem 1: Three Ways to Sum to N

Path: [`src/problem1`](src/problem1)

This problem implements three different ways to calculate the sum from `1` to `n`:

- using a `for` loop
- using recursion
- using the math formula `n * (n + 1) / 2`

Run it with:

```bash
cd src/problem1
node script.js
```

### Problem 2: Currency Swap

Path: [`src/problem2`](src/problem2)

This is a small React + TypeScript currency swap app. It includes:

- token price fetching
- token selection
- swap amount calculation
- mocked submit flow
- rates comparison tab
- dark and light theme support

Run it with:

```bash
cd src/problem2
npm install
npm run dev
```

### Problem 3: Messy React

Path: [`src/problem3`](src/problem3)

This problem reviews and refactors a messy React component. The refactor fixes:

- wrong TypeScript types
- incorrect filter logic
- unstable React keys
- unused formatted data
- unnecessary memo dependencies
- missing price fallback handling
